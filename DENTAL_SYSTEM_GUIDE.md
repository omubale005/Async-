# DentalCare - Clinic Management System

A modern, professional SaaS dashboard for dental clinic management used by dentists, receptionists, and clinic staff.

## Features

### 🏠 Dashboard
- Today's appointment schedule with timeline
- Quick statistics (patients, revenue, pending payments)
- Quick action buttons for common tasks
- Recent patients overview
- Role-based content display

### 👥 Patients Management
- Searchable patient database
- Detailed patient profiles with:
  - Personal information
  - Medical history
  - Visit timeline
  - Contact details
- Patient profile modal with full details

### 📅 Appointments
- Calendar view (Day/Week toggle)
- Color-coded appointment status
- Time-slot based daily schedule
- Weekly overview grid
- Appointment statistics

### 🦷 Dental Chart
- Interactive 32-teeth diagram
- Click to select individual teeth
- Color-coded conditions:
  - Healthy (Gray)
  - Cavity (Orange)
  - Filling (Green)
  - Crown (Blue)
  - Root Canal (Purple)
  - Missing (Red)
- Tooth details panel with notes
- Treatment summary statistics
- Patient selector

### 💰 Billing & Invoices
- Invoice creation form
- Dynamic invoice items
- Payment status tracking (Paid/Unpaid/Partial)
- Revenue statistics
- Pending payments overview
- Invoice list with download options

### 📦 Inventory Management
- Stock level tracking with progress bars
- Low stock alerts
- Category-based organization
- Restock history
- Visual stock indicators
- Category breakdown

### 📊 Reports & Analytics
- Revenue trend line chart
- Patient visits bar chart
- Treatment distribution pie chart
- Appointment status breakdown
- Performance metrics
- Export functionality
- Time period selector

### ⚙️ Settings
- Profile management
- Notification preferences
- Security settings
- Clinic information
- Operating hours configuration

## Role-Based Access

Switch between three user roles in the header dropdown:

1. **Dentist** - Full access to clinical features
2. **Receptionist** - Focus on appointments and billing
3. **Admin** - System configuration and reports

## Design Features

- ✨ Clean, minimal, modern SaaS design
- 🎨 Light theme with soft blue accents
- 📱 Responsive layout (desktop-first)
- 🔄 Smooth animations and transitions
- 🎯 Intuitive navigation
- ⚡ Fast loading indicators
- 🎨 Consistent design system
- 💫 Hover effects and micro-interactions

## Technology Stack

- React with TypeScript
- React Router for navigation
- Tailwind CSS for styling
- Recharts for analytics
- Motion for animations
- Radix UI components
- Lucide React icons

## Navigation

- Dashboard - Overview and quick actions
- Patients - Patient management and profiles
- Appointments - Calendar and scheduling
- Dental Chart - Interactive tooth diagram
- Billing - Invoice creation and tracking
- Inventory - Stock management
- Reports - Analytics and insights
- Settings - System configuration

## Quick Start

The application uses mock data for demonstration. All features are fully functional with sample patients, appointments, invoices, and inventory items.

To switch user roles, click on your profile in the top-right corner and select "Switch Role (Demo)".
